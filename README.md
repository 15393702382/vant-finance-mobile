# vant-finance-mobile
一个基于 [Vant](https://youzan.github.io/vant/) 的UI Demo，主要是为了学习和记录基于 Vue/Vant 开发移动设备单页应用的相关知识。

### 详情
记录和学习一些使用Vue来快速开发移动端HTML5页面的知识，主要包含：
* HTML+CSS基本布局的使用
* 基于Vue/Webpack开发，Vue模板，状态等的使用
* VueRouter 做页面路由跳转，参数传递等使用
* Vuex 做全局状态管理和同步等操作
* VueI18n 做页面的多语言支持
* Vant 各种组件的组合与使用
* Axios 做网络请求操作(Demo项目暂时未使用)
